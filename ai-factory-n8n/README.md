# n8n Workspace

This directory contains the public n8n/demo workspace used by the portfolio version of the project.

## Folders

- `library/`: empty placeholder folders for styles, characters, outfits, poses, expressions, cameras, and backgrounds
- `presets/`: empty placeholder folder kept for structure only
- `queue/`: the active queue payload consumed by the workflow
- `n8n_workflow_v6.json`: importable workflow definition

## Demo content policy

The portfolio copy keeps only the folder structure for `library/` and `presets/`. Private assets, prompts, and sensitive content are intentionally excluded.
